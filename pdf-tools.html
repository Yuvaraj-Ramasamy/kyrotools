Frequently Asked Questions

How are files merged without server uploads?
We load your PDFs as raw binary bytes directly into the local browser. The pdf-lib JavaScript package compiles the binary segments, copies pages into new structures, and outputs a downloadable file in the browser memory.
What is the split range parameter syntax?
To extract pages, define targets separated by commas: e.g., 1-3 splits pages 1, 2, and 3; typing 5 adds page 5; and 8-10 appends pages 8, 9, 10. You can type single page numbers or page ranges together.
Is there a maximum file size limit?
Because compilation relies on your local device RAM, most standard devices easily handle documents up to 100MB or 200 pages without any lags. Large files process in milliseconds.
